Reflections

Event bubbling means that after running any handlers on the element that first triggered it,
the event will 'bubble' upwards, that is travel up/out to its parent element, then its parent etc all the way to the document as a whole.
On each new element it will trigger any eventhandlers there. We can use this to use a single eventhandler on the outer div to write messages from each of the ten my-divs.
